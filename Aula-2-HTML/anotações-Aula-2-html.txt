 O QUE SÃO TAGS?

Tags são como *etiquetas* usadas para informar ao navegador como o conteúdo deve ser apresentado em uma página da web.

Quando escrevemos tags no HTML, o navegador *lê e interpreta* essas instruções para montar o site visualmente.

---------------------------------------------------
💡 COMO REPRESENTAR UMA TAG?

As tags são escritas assim:

<tag>conteúdo</tag>

Por exemplo:
<p>Esse é um parágrafo.</p>
<h1>Título principal</h1>

As tags são sempre envolvidas por sinais de "menor que" e "maior que":
< e >

---------------------------------------------------
📎 E A TAG <code>?

A tag <code></code> é usada quando queremos mostrar um trecho de código na página.

Exemplo:
<p>Use o comando <code>print("Olá")</code> para imprimir no Python.</p>

Isso aparece no navegador como:
Use o comando print("Olá") para imprimir no Python.

Ela *não executa* o código. Só mostra o texto com *formato de código*, como se fosse um editor.

---------------------------------------------------
🎯 CADA TAG TEM UM PROPÓSITO:

- <a> → Link
- <p> → Parágrafo
- <ul> e <li> → Listagem
- <img> → Imagem
- <h1> até <h6> → Títulos
- <br> → Quebra de linha
- <code> → Mostrar código
- E muitas outras...

---------------------------------------------------
✅ DICA FINAL:
- O navegador *não mostra as tags, ele usa elas para saber **como mostrar* o conteúdo.
- Aprender as tags básicas é o primeiro passo para criar sites!
